>包含Vim的配置文件、添加Python和Luau支持的Vim可执行程序

#前提
1. Vim 8.0(+) with Vundle
2. 安装字体补丁
3. 安装Python 2.7(+)、Python 3.5(+)、Lua 5.1(+)

#安装
1. 备份配置文件
2. git clone https://github.com/netgather/vim-config.git
3. 替换_vimrc
4. 替换gvim.exe
5. 运行命令：PluginInstall
